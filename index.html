Central de Ajuda & Guia Rápido

Bem-vindo ao Mission Control da MetalLAB. Esta ferramenta foi desenhada para gerir todo o ciclo de vida do robô: desde o tracking de partidas até à engenharia e análise de desempenho.

Match Tracker

Registe os pontos de cada partida em tempo real.

  • Use os contadores +/- para ações (Classified, Overflow, etc.)
  • Selecione a configuração do robô no dropdown (puxado dos Blueprints).
  • Marque os problemas ocorridos para análise posterior.
  • Clique em Sincronizar Partida para guardar na nuvem.
Dica: A eficiência é calculada automaticamente com base na meta definida.

DevOps Log

Documente todas as alterações de hardware e software.

  • Cada ticket deve ter um subsistema, versão e título.
  • Preencha a telemetria (KPI antes/depois) para medir o impacto.
  • Descreva a hipótese, solução e resultados (RCA).
  • O custo inserido alimenta automaticamente o BOM do Blueprint.
Dica: Use o botão "Puxar Dados da Nuvem" no Blueprint para importar as últimas versões.

Blueprint & BOM

Visualize e atualize a configuração atual do robô.

  • Preencha o nome e versão no topo.
  • Os campos de cada subsistema podem ser preenchidos manualmente ou via botão "Puxar Dados".
  • O custo total é calculado automaticamente.
  • Guarde o Blueprint para criar uma nova versão comparável.

Comparar Versões

Analise a evolução entre dois Blueprints guardados.

  • Selecione duas versões nos dropdowns.
  • A tabela destaca diferenças: verde para melhoria, vermelho para piora.
  • Use para justificar alterações de design.

Analytics

Visualize o desempenho da equipa com gráficos interativos.

  • O histórico mostra a evolução da pontuação com média móvel.
  • Gráficos de dispersão e pizza detalham a contribuição de cada ação.
  • A tabela de partidas permite excluir registos errados.
  • Use o botão "Atualizar Dados" para recarregar os gráficos.

Perguntas Frequentes (FAQ)

Como corrijo uma partida registada com erro?

Na aba Analytics, localize a partida na tabela e clique no ícone de lixeira para excluí-la. Depois, regresse ao Match Tracker e registe novamente.

O que significa "Consistência (CV)"?

É o Coeficiente de Variação (desvio padrão / média). Quanto menor o valor, mais consistente é o desempenho do robô. Valores abaixo de 20% são considerados bons.

Posso usar a ferramenta offline?

Sim, os dados são guardados localmente durante a sessão e sincronizados com o Firebase assim que houver conexão. As alterações aparecerão em todos os dispositivos ligados à mesma conta Firebase.

Como funciona o preenchimento automático do Blueprint?

O botão "Puxar Dados da Nuvem" procura os commits mais recentes no DevOps Log para cada subsistema (DRV, SEN, MEC_INT, etc.) e preenche a versão e o custo associado.

Por que os gráficos não aparecem?

Certifique-se de que existem partidas registadas. Se ainda assim não aparecerem, clique em "Atualizar Dados" na aba Analytics ou mude de tema (claro/escuro) para forçar a renderização.

Como mudo o tema para escuro?

Clique no ícone de lua/sol no canto superior direito. A preferência é guardada automaticamente.
